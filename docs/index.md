# 不動産業界インプット読本

不動産業界向けAIエージェント導入コンサルを始めるにあたって、業界全体を素早く理解するための読本集です。15冊の書籍をテーマ別に再編・リライトしています。

---

## 収録している読本

| Vol | タイトル | 内容 |
|-----|---------|------|
| [Vol.1](vol1.md) | 業界の全体地図 | 市場規模・ビジネスモデル・プレーヤー・法規制 |
| [Vol.2](vol2.md) | 業界人のリアル | 営業の一日・職種の違い・業界文化・給与 |
| [Vol.3](vol3.md) | テクノロジーが変える不動産業 | PropTech・AI活用・デジタル化の現在地 |
| [Vol.4](vol4.md) | 消費者から見た不動産取引 | 購入プロセス・よくある罠・詐欺の歴史 |
| [Vol.5](vol5.md) | 業界の面白話と裏側 | クソ物件・倒産あるある・宅建の法律感覚 |

---

## 使い方

- **上部タブ**または**左のサイドバー**からVolを選んで読む
- **右上の検索ボックス**で全Vol横断検索できる
- **スマホ**の場合は左上のメニューアイコンからナビゲーション
- **ダークモード**は右上の太陽/月アイコンで切り替え
